# Legal pages (GitHub Pages)

Published from the `docs/` folder for App Store Connect.

## URLs (after Pages is enabled)

- **Support / home:** https://noamtzur21.github.io/home-nas-player/
- **Privacy Policy:** https://noamtzur21.github.io/home-nas-player/privacy-policy.html
- **Terms of Service:** https://noamtzur21.github.io/home-nas-player/terms-of-service.html

## One-time setup (GitHub website)

1. Open https://github.com/noamtzur21/home-nas-player/settings/pages
2. **Build and deployment → Source:** choose **GitHub Actions**
3. Push to `main` or run workflow **Deploy legal pages** manually
4. Wait ~1 minute, then open the Privacy URL above
